*Running Fest UNESA*
📖 Tentang Proyek
Website ini dirancang sebagai pusat informasi permanen untuk menumbuhkan gaya hidup sehat di lingkungan kampus UNESA. Bukan hanya sekadar ajang lomba tahunan, website ini mendukung keberlanjutan komunitas melalui edukasi rencana latihan (training plans).

✨ Fitur Utama
- Landing Page Responsif: Tampilan yang menyesuaikan dengan perangkat mobile maupun desktop.
- Program Latihan: Rencana lari terstruktur untuk kategori Beginner 5K, Intermediate 10K, dan Endurance Challenge.
- Galeri Event: Dokumentasi kegiatan lari yang telah dilaksanakan.
- Sistem Pendaftaran Modal: Integrasi syarat & ketentuan event dengan tautan langsung ke Google Form pendaftaran.
- Smooth Scrolling: Navigasi antar bagian halaman yang halus untuk pengalaman pengguna yang lebih baik.

🛠 Teknologi yang Digunakan
- HTML5: Struktur konten utama.
- CSS3: Styling kustom (termasuk penggunaan framework internal framework.css).
- JavaScript (Vanilla): Logic untuk interaksi modal pendaftaran dan navigasi.
- FontAwesome: Ikon pendukung antarmuka.
